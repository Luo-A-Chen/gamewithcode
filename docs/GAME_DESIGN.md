# 《代码编年史》游戏设计规范 (Game Design Document)

> 本文档是游戏开发的核心参考，所有开发工作应以此为准。

## 基本信息

- **标题**：代码编年史 (Code Chronicle)
- **类型**：2D 平台跳跃 + 教育
- **技术栈**：HTML5 Canvas + ES Modules，零依赖
- **目标**：通过游戏机制隐性传递编程历史与概念

---

## 目标受众

| 维度 | 描述 |
|------|------|
| **核心受众** | 对编程感兴趣的青少年和成人（15-35岁）|
| **次要受众** | 2D 平台跳跃爱好者、独立游戏玩家 |
| **教育场景** | 编程入门课程的辅助学习材料 |
| **玩家画像** | "想了解编程历史但不想看教科书的人" |

---

## 独特卖点 (USP)

1. **编程历史 × 平台跳跃**：用游戏机制隐性教学，不是说教
2. **7个时代、7种机制**：每个世界的核心玩法都对应一个编程概念
3. **真实历史**：Boss 是真实编程问题的拟人化，NPC 是真实历史人物
4. **零门槛**：浏览器直接运行，无需安装

## 一句话概述

一名程序员意外被吸入计算机历史的时间线，必须穿越编程发展的每个重要时代，修复被"Bug 军团"篡改的历史节点，才能回到现实。

---

## 世界观

在计算机的世界里，存在一条"代码之河"——串联所有编程语言和技术诞生的时间线。一个名为 **Bug King** 的存在正在腐蚀这条河流，试图让编程历史倒退，让世界永远停留在机械计算的时代。

玩家扮演的 **Coder**（一个普通程序员）在 debug 时意外触发了时间裂缝，被卷入代码之河。必须穿越 **7 个时代**，收集每个时代的核心"语言碎片"，击败被腐化的守护者，才能重塑时间线。

---

## 7 个世界

### 世界1：机械之心（1840s 巴贝奇差分机）
- **视觉**：铜色齿轮、蒸汽管道、黄铜色调
- **机制**：基础跳跃、推箱子（齿轮拼合）
- **Boss**：被卡住的差分机
- **NPC**：巴贝奇教你齿轮逻辑
- **教育点**：机械计算、条件逻辑

### 世界2：电子黎明（1940s ENIAC）
- **视觉**：真空管、电缆迷宫、闪烁灯光
- **机制**：电路开关（踩按钮开门）
- **Boss**：暴走的继电器
- **NPC**：图灵的影子
- **教育点**：二进制、开关逻辑

### 世界3：语言摇篮（1950-60s FORTRAN/LISP）
- **视觉**：黑底绿字、打孔卡片雨
- **机制**：收集"打孔卡片"拼成语句通过障碍
- **Boss**：无限递归怪物（栈溢出），需找到递归出口
- **NPC**：Grace Hopper 教你"debug"的由来
- **教育点**：递归、终止条件、语法

### 世界4：结构之光（1970s C/Unix）
- **视觉**：管道符连接的平台、终端界面
- **机制**：管道传送、文件系统导航
- **Boss**：野指针幽灵（瞬移攻击）
- **NPC**：Dennis Ritchie 给你 C 语言之力
- **教育点**：指针、内存、Unix 管道

### 世界5：面向对象城（1980-90s C++/Java）
- **视觉**：城市建筑、类图结构、UML 风格
- **机制**：继承/组合机制（复制能力）
- **Boss**：内存泄漏黑洞（不断膨胀吞噬）
- **NPC**：Bjarne Stroustrup
- **教育点**：继承、多态、封装

### 世界6：互联纪元（2000s Web/JS）
- **视觉**：网页节点、超链接跳跃、HTML 标签
- **机制**：超链接传送、协议解谜
- **Boss**：XSS 攻击虫群
- **NPC**：Tim Berners-Lee 打开超链接之门
- **教育点**：URL 路由、HTTP、安全

### 世界7：智能纪元（2010s+ AI/深度学习）
- **视觉**：神经网络、数据流瀑布、矩阵
- **机制**：AI 辅助（NPC 自动搭路）
- **Boss**：Bug King 终极形态
- **NPC**：现代 AI 研究者
- **教育点**：神经网络、训练与推理

## 角色设计

### 主角：Coder
- 外形：像素风小人，耳机 + 连帽衫
- 能力随世界进化，每个世界获得新技能

### 反派：Bug 军团
每个 Boss 是该时代代表性编程问题的拟人化：
- 被卡住的差分机 → 逻辑错误
- 暴走的继电器 → 硬件故障
- 无限递归怪物 → 栈溢出
- 野指针幽灵 → 空指针
- 内存泄漏黑洞 → 内存泄漏
- XSS 虫群 → 安全漏洞
- Bug King → 所有 Bug 的集合体

### NPC：编程先驱
每个世界有一位历史人物作为引导者。

## 收集系统

### 语言碎片（每关 3 个）
收集后解锁该时代编程语言简介。示例：

> "1972 年，Dennis Ritchie 在贝尔实验室创造了 C 语言。
> 它像一把手术刀——精准、锋利，也容易伤到自己。"

### Bug 日志（隐藏收集）
记录真实历史上的著名 Bug：
- 1962 水手一号（连字符缺失）
- 1990 AT&T 网络瘫痪（代码缺陷）
- 2014 Heartbleed（缓冲区溢出）

## 机制 ↔ 编程概念映射

| 游戏机制 | 编程概念 |
|---------|---------|
| 踩按钮开门 | 条件判断 (if/else) |
| 管道传送 | Unix 管道 `|` |
| 复制能力 | 继承与多态 |
| 超链接传送 | URL 路由 |
| 收集卡片拼语句 | 语法与逻辑 |
| 递归怪物 | 递归与终止条件 |
| 推齿轮 | 算法流程 |
| 电路开关 | 二进制与布尔逻辑 |

## 叙事节奏

```
开场：程序员深夜 debug → 屏幕闪烁 → 被吸入时间线
  ↓
世界1-2：认识 Bug 军团，了解代码之河的存在
  ↓
世界3-4：发现 Bug King 的阴谋——篡改编程语言诞生
  ↓
世界5-6：获得关键能力，准备最终决战
  ↓
世界7：面对 Bug King，用所有学到的知识修复时间线
  ↓
结局：回到现实，屏幕上多了一行注释：
      // 你以为在 debug，其实是在书写历史。
```

## 当前技术架构

```
src/
├── core/          # 引擎核心（Game、Input、Camera、Collision、Renderer）
├── entities/      # 实体系统（Entity基类、Player、Enemy、Collectible）
├── levels/        # 关卡系统（Level、LevelManager）
├── ui/            # UI 组件（预留）
└── utils/         # 工具函数
```

关键设计决策：
- 纯 ES Modules，零依赖
- AABB 瓦片碰撞 + 最小穿透分离
- requestAnimationFrame + delta time
- 实体继承体系 + update/render/onCollide 接口
